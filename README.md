
# Infra Playground

이 프로젝트는  
Docker, docker-compose, Redis, Kubernetes, CI/CD를 활용하여  
AI 모델의 훈련 환경과 검증 환경을 직접 구축해보는 스터디 레포지토리입니다.

목표는 단순합니다.
 
> 모델을 잘 만드는 것보다  
> 모델이 잘 돌아가는 환경을 만드는 것.

---

## 🧱 스터디 목표

- Docker 기반 학습 환경 구성
- docker-compose로 멀티 서비스 구성
- Redis를 활용한 캐시/큐 구조 이해
- Kubernetes 기반 컨테이너 오케스트레이션 이해
- CI/CD를 통한 자동 학습/테스트 파이프라인 구성
- 훈련 환경과 검증 환경의 분리 및 재현성 확보
- 부트캠프 최종 프로젝트에서 스터디서 배운내용 활용

---

## ⚙️ 우리가 다루는 기술

- Docker
- docker-compose
- Redis
- Kubernetes (k8s)
- GitHub Actions 기반 CI/CD
- ML / DL 모델 학습 코드

## 🔬 우리가 실험하는 것

---

## 🚀 스터디 방식

1. 로컬에서 직접 실행
2. Docker 환경에서 실행
3. docker-compose로 서비스 구성
4. k8s로 배포 구조 구성
5. CI/CD로 자동 테스트 및 검증

이 레포는 정답 코드가 아니라  
환경을 이해하고 직접 구축해보는 실험 공간입니다.
